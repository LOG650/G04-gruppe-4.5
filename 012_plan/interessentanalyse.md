# Interessentanalyse: Lageroptimalisering Skoringen Råholt

Dette dokumentet kartlegger aktørene som har interesse i prosjektet og hvordan de bør håndteres.

| Interessent | Interesse (1-5) | Påvirkning (1-5) | Strategi |
| :--- | :---: | :---: | :--- |
| **Daglig leder (Eier)** | 5 | 5 | Nøkkelperson. Må involveres i alle beslutninger og se den økonomiske gevinsten. |
| **Butikkansatte** | 3 | 4 | Sluttbrukere. Må få opplæring og forstå at systemet forenkler deres hverdag (mindre tunge løft til eksternt lager). |
| **Skoringen-kjeden (Sentralt)** | 2 | 2 | Informeres om resultater. Kan være interessert i å rulle ut modellen til flere butikker. |
| **Leverandører / Grossister** | 3 | 3 | Dialog om hyppigere leveranser. Trenger forutsigbarhet i bestillingene. |
| **Kunder** | 4 | 2 | Indirekte berørt. Opplever bedre vareutvalg og færre "utsolgt"-situasjoner. |

## Interessent-matrise
- **Høy interesse / Høy makt:** Daglig leder (Administrere tett).
- **Høy interesse / Lav makt:** Kunder (Holdes orientert via vareutvalg).
- **Lav interesse / Høy makt:** Butikkansatte (Holdes tilfreds via god brukervennlighet).
