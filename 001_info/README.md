# 001_info – Prosjektinformasjon

Denne mappen samler all informasjon som ikke hører hjemme i et faglig leveransekapittel, men som er viktig for prosjektgruppen å ha lett tilgjengelig.

## Innhold

| Fil | Formål |
|---|---|
| `kontakter.md` | Kontaktinformasjon for gruppemedlemmer, veiledere og kunde |
| `skoringen_raholt.md` | Bedriftsinformasjon om casebedriften |
| `beslutningslogg.md` | Logg over viktige beslutninger som er tatt underveis |

## Bruk

- **Kontakter** oppdateres når noe endres (telefon, e-post, rolle).
- **Beslutningsloggen** oppdateres når gruppen tar en faglig eller praktisk beslutning som kan være viktig å huske eller forklare senere (f.eks. valg av modell, scope-endring, endring i metode). Hver oppføring bør ha dato, beslutning, begrunnelse og hvem som var involvert.
- **Skoringen Råholt-fil** brukes som referanse i rapporten – ikke som duplikat, men som kilde for fakta som ellers ville måttet hentes hver gang.
