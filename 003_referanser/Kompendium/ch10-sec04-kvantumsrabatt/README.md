# Quantity Discount EOQ

Example code for sec04 of Ch10: classical EOQ extended with quantity-discount
price schedules, implemented for both all-units and incremental discount models.

Run in order:

```
uv sync
uv run python src/step01_datainnsamling.py
uv run python src/step02_basic_eoq.py
uv run python src/step03_all_units.py
uv run python src/step04_incremental.py
uv run python src/step05_sammenligning.py
uv run python src/step06_anbefaling.py
```
